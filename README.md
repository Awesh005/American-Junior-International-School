> **"Nurturing young minds with love, care, and quality education."**

Welcome to the official repository for the **American Junior International School** website. This project is a modern, high-performance web application designed to provide a joyful and informative experience for parents and students.

---

## ✨ Key Features

- 🎨 **Playful & Modern UI**: A vibrant, kid-friendly design using pastel colors and rounded elements.
- 📱 **Fully Responsive**: Seamless experience across mobile, tablet, and desktop devices.
- 🎭 **Smooth Animations**: Powered by `framer-motion` for entrance reveals, floating shapes, and interactive hover states.
- 🧭 **Intuitive Navigation**: Clear access to Programs, Activities, Gallery, and Admissions.
- 📝 **Admission Portal**: A structured multi-step admission process and inquiry system.
- 📍 **Interactive Contact**: Integrated Google Maps and direct WhatsApp/Call links.
- 🚀 **SEO Optimized**: Meta tags, robots.txt, and clean semantic HTML for better search engine visibility.
- 🛡️ **Production Ready**: Includes Error Boundaries and a custom 404 page for a robust user experience.

---

## 🛠️ Tech Stack

- **Frontend Framework**: [React 18](https://reactjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router Dom v7](https://reactrouter.com/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/ajis-website.git
   cd ajis-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
